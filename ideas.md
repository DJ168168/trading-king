## 方案一：赛博朋克终端风格 (Cyberpunk Terminal)

**设计运动**：赛博朋克 + 终端美学，灵感来自《银翼杀手》和黑客帝国的视觉语言

**核心原则**：
1. 深邃黑底配合霓虹绿/红作为主要信号色，营造专业交易终端氛围
2. 网格化信息密度，最大化数据展示效率
3. 扫描线和发光效果强化科技感

**色彩哲学**：
- 背景：纯黑 #0a0a0a 到深灰 #111111 的微妙渐变
- 主色：霓虹绿 #00ff88 代表盈利/做多
- 警告色：霓虹红 #ff4444 代表亏损/做空
- 辅助色：琥珀黄 #ffaa00 代表警告/中性
- 文字：冷白 #e0e0e0 主文字，#888888 次要文字

**布局范式**：固定左侧导航 + 右侧内容区的经典交易终端布局，内容区使用卡片网格系统

**签名元素**：
1. 卡片边框使用 1px 绿色/红色发光线条
2. 数据变化时的脉冲动画效果
3. 终端风格的等宽字体数字显示

**交互哲学**：快速响应，数据驱动，hover 时边框发光增强

**动画**：数字跳动效果、卡片淡入、状态切换时的微妙闪烁

**字体系统**：JetBrains Mono 用于数字和代码，Inter 用于界面文字

## 方案二：暗黑玻璃拟态风格 (Dark Glassmorphism)

**设计运动**：玻璃拟态 + 暗色模式，灵感来自 Apple Vision Pro 和现代金融终端

**核心原则**：
1. 半透明毛玻璃卡片叠加在深色背景上，创造层次感
2. 柔和的光影和模糊效果替代硬边框
3. 精致的微动画提升高端感

**色彩哲学**：
- 背景：深空黑 #050510 带微妙蓝色调
- 卡片：rgba(255,255,255,0.05) 半透明 + backdrop-blur
- 主色：翡翠绿 #10b981 代表积极信号
- 危险色：玫瑰红 #ef4444 代表风险
- 强调色：电光蓝 #3b82f6 用于交互元素

**布局范式**：侧边导航使用毛玻璃效果，内容区卡片浮动在深色背景上

**签名元素**：
1. 所有卡片使用 backdrop-blur-xl 毛玻璃效果
2. 渐变色边框（从绿到蓝的微妙过渡）
3. 悬浮时的光晕扩散效果

**交互哲学**：优雅流畅，每个交互都有丝滑的过渡动画

**动画**：卡片入场使用 spring 弹性动画，数据更新使用淡入淡出

**字体系统**：Space Grotesk 用于标题，DM Sans 用于正文，Fira Code 用于数字

## 方案三：军事级指挥中心风格 (Command Center)

**设计运动**：军事指挥中心 + 数据可视化仪表盘，灵感来自 NASA 任务控制和军事 HUD

**核心原则**：
1. 高信息密度但层次分明，用颜色编码区分数据优先级
2. 硬朗的直角边框和网格线强化严肃专业感
3. 状态指示灯和实时数据流营造紧迫感

**色彩哲学**：
- 背景：军事深灰 #0d1117 到 #161b22
- 主色：战术绿 #00d97e 代表正常/盈利
- 警报色：战术红 #ff3b3b 代表危险/亏损
- 信息色：战术蓝 #58a6ff 代表信息/中性
- 高亮色：战术橙 #f0883e 代表警告

**布局范式**：左侧任务面板 + 中央主显示区 + 状态栏，模拟指挥中心多屏布局

**签名元素**：
1. 顶部状态栏显示系统运行状态（类似 HUD）
2. 卡片左侧带有颜色编码的状态条
3. 数据表格使用交替行高亮和扫描线效果

**交互哲学**：精确高效，最少点击达到目标，键盘快捷键支持

**动画**：数据刷新时的扫描线效果，新信号到达时的脉冲提醒

**字体系统**：Rajdhani 用于标题和数字（军事感），Source Sans 3 用于正文
